# cadac
Lernprogramm für Französisch. Während der Oberstufe (2006-2010) habe ich das Voci für den Franzunterricht am Computer lernen wollen.
Leider gab es kein passendes Lernprogramm für mich. Daher hab ich es selber geschrieben in der Sprache `Delphi` (basiert auf Pascal).

### Funfact
Das Programm läuft im Jahr 2019 mit Windows 10 - mehr oder weniger - noch immer.  
Bekannte Einschränkungen sind:
- Es gibt keine Updates mehr
- Wenn der Dokumente-Ordner verschoben wurde, klappt die Installation nicht
- Kein Drucker-Dialog, die Wörterlisten landen alle auf dem Standard-Drucker
- Einige Felder "fehlen" (Beschriftungen werden nicht mehr angezeigt)
- ...
